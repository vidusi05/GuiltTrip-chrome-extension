Private Chrome Extension for PeraCom Students going through exams.
Note: All Media are available publicly on people.ce.pdn.ac.lk pages. 

Instructions: 
1. Download the repository on your local machine
2. Visit chrome://extensions/ on your browser
3. Enable developer mode
4. Select "Load Unpacked" on the top left hand side.
5. Select the repo
6. Extension will automatically be enabled. Visit youtube and enjoy (sort of)
